Frequently Asked questions.

The DOM (Document Object Model) is an HTML elements structure in a tree format. With Javascript, we can select the HTML elements and update their properties. DOM lets us change attributes, styles, text content, etc. using Javascript on the go.

There are multiple different APIs to select an element from the dom like,

document.getElementById()
document.getElementsByClassName()
document.getElementsByTagName()
document.querySelector(), document.querySelectorAll()

Suppose we need to add event listeners to multiple elements. Despite adding an event listener to an individual item, we can add an event listener to the parent of those elements and detect which specific element was clicked.
This way, we don't need to select many elements, so adding just one event listener works fine.

Many browser APIs available for manipulating an element's attributes and styles.

## Here's how to get the value of an attribute.
const element = document.getElementById('myElement')
const hrefValue = element.getAttribute('href')

## Here's how to set an attribute.
const element = document.getElementById('myElement')
element.setAttribute('href', 'https://new-link.com')

To manipulate styles, we can directly do it by using
const element = document.getElementById('myElement')
element.style.backgroundColor = 'red'
Or we can do it by setting and removing some classes
const element = document.getElementById('myElement')
element.classList.remove()
element.classList.add()
